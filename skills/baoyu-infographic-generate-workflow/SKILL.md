# Baoyu Infographic Generate Workflow

This skill automates the infographic generation process with predefined settings:
- **Default layout**: dense-modules
- **Default style**: morandi-journal  
- **Default aspect**: 9:16 (portrait)
- **Default language**: Korean (ko)
- **Output directory**: /Users/kwaksmacmini/hermes-infographic/

## Usage

Run this skill to generate infographics with consistent settings and automatic file management.

## Key Features

- Saves to `/Users/kwaksmacmini/hermes-infographic/`
- Uses Korean language by default
- Applies dense-modules layout with morandi-journal style
- Maintains consistency across all generated infographics
- No trial-and-error needed - all settings are pre-configured
